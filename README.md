# BoonBuy Finds

Independent finds catalog for **BoonBuy** — cloned from the LitBuy Finds stack with BoonBuy branding, domains, and agent links.

## Domains

- **Primary:** https://boonbuyfinds.net
- **Redirect:** https://boonbuys.com → boonbuyfinds.net

## Agent

- Recommended agent: **BoonBuy** (https://boonbuy.com)
- Signup / coupon invite: `src/lib/boonbuy-affiliate.ts` (`inviteCode=BOONFINDS`)
- Social links: `SOCIAL_LINKS` in `src/lib/constants.ts` (Telegram `@RNFinds`, Discord, Instagram, TikTok)

## Develop

npm install && npm run dev
